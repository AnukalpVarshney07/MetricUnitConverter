## 📌 Description

**Unit Convert** is a responsive web-based application built using HTML, CSS, and JavaScript that performs fast and accurate unit conversions. It supports multiple categories like length, mass, temperature, time, and speed with real-time results and a clean, user-friendly interface.

---

## ✨ Features

- 🔄 Multi-category conversions  
- ⚡ Real-time calculation  
- 🎯 High accuracy  
- 🎨 Clean & modern UI  
- 📱 Fully responsive
